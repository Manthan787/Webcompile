Webcompile

Online Compiler for C, C++ and Java!

Download as .zip Download as .tar.gz View on GitHub

An online compiler for your web applications!

Webcompile allows you to compile and run programs written in C, C++ and Java Programming languages right from your web application. This package is ideal for creating online code editors and other related applications.

Install

{
  require: {
        "manthan/webcompile" : "0.3"
  }
}
